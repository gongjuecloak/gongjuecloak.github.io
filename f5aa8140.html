putchar

  • int putchar(int c);
  • 向标准输出写一个字符
  • 返回写了几个字符，EOF(-1)表示写失败

getchar

  • int getchar(void)
  • 从标准输入读入一个字符
  • 返回类型时int是为了返回EOF(-1)
  • windows——-ctrl+Z
  • linux—————-ctrl+D

strcpy

  • char strcpy(char restrict dst, const char *restrict src);
  • 把src的字符串拷贝到dst
  • restrict表明src和dst不重叠（C99）
  • 返回dst，为了能链起代码
